memo
================================================================================

定番置換
--------------------------------------------------------------------------------
```vim
:%s/hoge/piyo/cg
```

-最初のコロン → コマンドモード開始
- `%s` → 全行対象
- `hoge` → 検索対象の正規表現
- `piyo` → 検索合致モノをpiyoで置換
- `c` → 置換時に確認アリ
- `g` → 最初の一回でなく全部


クリップボードからのペースト
--------------------------------------------------------------------------------
` Shift + Insert`



カーソル位置の単語で検索
--------------------------------------------------------------------------------
ノーマルモードでアスタリスク `*` を押すとカーソルがある位置の単語で検索できる。便利！
