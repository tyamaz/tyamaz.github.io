フック時に設定する
--------------------------------------------------------------------------------
`Firefox` のイベントモデルではイベントがバブリングフェーズで反応するかどうかを明示的に選択することができる


発生時にバブリングを抑制する
--------------------------------------------------------------------------------
`IE` ではフック時に設定できないので、発生時に伝播を抑制する。`Firefox` でも同じようなことができる

Firefox

```javascript
e.stopPropagation();
```

IE   

```javascript
window.event.cancelBubble = true;
```
