Silverlight 1.0, 1.1, 2.0 の違い
================================================================================
`1.0` と `1.1` の間で全然違う。全然別の技術

`1.0` はお手軽に JavaScript で制御できるおもちゃが HTML 以外にも広がった感じで俺は好感が持てたが、だが `1.1` 以降は ASP.NET っぽくなってしまってダメダメ。

`2.0` とかは開発環境がグダグダでどうもしっくりこない
