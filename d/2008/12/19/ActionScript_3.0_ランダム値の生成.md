- 2008年12月19日
- ActionScript3.0

ランダム値を生成　小数版
================================================================================

0 以上 1 未満のランダム値を生成する
--------------------------------------------------------------------------------

```actionscript
trace(Math.random());
```


0以上2未満のランダム値を生成する
--------------------------------------------------------------------------------
```actionscript
trace(Math.random() * 2);
```

ランダム値を生成　整数版
================================================================================
0〜9のランダム値を生成する
--------------------------------------------------------------------------------
```actionscript
trace(Math.floor(Math.random() * 10));
```
