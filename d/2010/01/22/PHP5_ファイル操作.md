ファイルの存在確認をする
================================================================================
`file_exists` 関数でできる。これはディレクトリに対しても `true` を返す

```php
if(file_exists($hoge)){
    #なにか
}
```

ファイル内容操作系
================================================================================
テキストファイルを読み込んで一行ずつ読み込む
--------------------------------------------------------------------------------


ファイルパス操作系
================================================================================
ファイルパスからファイル名を抽出する
--------------------------------------------------------------------------------

```php
$path = "/hoge/piyo.txt";
$name = basename($path);
echo $name; #=> "piyo.txt"
```

`basename` 関数は、ディレクトリに対してやるとディレクトリ名がそのまま出るだけです・・・
なんか使い勝手が悪い。
