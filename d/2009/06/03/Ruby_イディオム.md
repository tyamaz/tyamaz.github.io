- 2009年06月03日
- Ruby 1.8.6

変数の中身がからっぽなら入れる
================================================================================

```ruby
a ||= "A"
```

↓と同じ

```ruby
unless a
  a = "A"
end
```

nil ならば nil 値が入っているならばそっちを採用
================================================================================

```ruby
a = nil
b = a && a.to_i

b #=> nil

a = "123"
b = a && a.to_i

b #=> 123
```
