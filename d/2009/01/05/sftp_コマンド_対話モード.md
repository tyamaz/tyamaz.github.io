接続
================================================================================
対話モードで接続する
--------------------------------------------------------------------------------

```console
$ sftp hogeuser@hogedomain
```

対話モードで接続する（ポート番号指定）
--------------------------------------------------------------------------------

```
$ sftp -oPort=12345  hogeuser@hogedomain
```


対話モード中
================================================================================
リモート側でディレクトリ移動
--------------------------------------------------------------------------------
ローカルでの場合と同じ

```
cd ディレクト名
```

リモート側でのディレクトリの中身表示
--------------------------------------------------------------------------------
ローカルでの場合と同じ

```
ls
```


ファイルのアップロード（ローカル→リモート）
--------------------------------------------------------------------------------

```
put ローカルファイル名 リモートファイル名
```
