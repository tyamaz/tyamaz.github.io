関数の括弧は省略できる
================================================================================
1文分だけならば関数の後の括弧は省略できる。if と一緒みたいね

```javascript
function hoge()
    alert("a");
```

関数リテラル
================================================================================
当然関数オブジェクトリテラルも作れる短い処理でチェーンするときとかはいいかもね。

```javascript
var hoge = function() alert("a");
```
